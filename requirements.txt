# 1. SYSTEM & PLATFORM REQUIREMENTS
# ------------------------------------------------------------------------------
Operating System: Linux / macOS / Windows (Docker recommended)
Docker Engine: >= 20.10.0
Docker Compose: >= 2.0.0
Node.js: >= 20.x (Recommended for local development)
NPM: >= 10.x
Database: SQLite 3

# ------------------------------------------------------------------------------
# 2. RUNTIME ENVIRONMENTS & PORTS
# ------------------------------------------------------------------------------
Frontend Port: 80 (Nginx Proxy)
Backend Port: 5000 (Express API Server)
Prisma SQLite Database Path:
  - Development: backend/dev.db
  - Container/Production: /app/data/prod.db (Persistent Volume: diemanager-db)

# 5. CORE FEATURES CAPABILITIES
# ------------------------------------------------------------------------------
- Universal Search: Multi-entity keyword search across Machines, Sets, and Dies.
- Range Search: Numeric parsing & filtration of string size attributes on Dies.
- Excel Bulk Import: Automated parsing and data upserting of `.xlsx`/`.xls` dies sheet.
- Audit Logging: Real-time action logs stored in SQLite database.
- Swagger API Docs: Interactive OpenAPI v3 specifications generated on `/api-docs`.
